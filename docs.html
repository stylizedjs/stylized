Ideally designed for beginning developers who need agility when developing application layouts, the project is completely open source and can be found on GitHub.

    Installation

    There is so many ways to start using Stylized in your project.

    CDN

    By far, the easiest way is using via CDN. Add the following line on your head HTML tags: 

    <link rel="stylesheet" href="https://unpkg.com/stylizedcss/src/stylized.css">
